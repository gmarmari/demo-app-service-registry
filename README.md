# demo-app-service-registry
Just a demo project.
The service registry of my demo spring application


 # Java
 It requires Java version 17

# Build the App

```
./mvnw clean package
./mvnw spring-boot:run
 ```
 
 
 ### Or with docker
 ```
 docker-compose up --build

 ```
